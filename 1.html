HTML

HYPERTEXT MARKUP LANGUAGE (HTML) IS THE STANDARD MARKUP LANGUAGE FOR CREATING WEB PAGES AND WEB APPLICATIONS.

WEB BROWERS RECEIV HEML DOCUMENTS FROM A WEB SERVER OR FROM LOCAL STORAGE AND RENDER THEM INTO MULIMEDIA WEB PAGES. HTML DESCRIBES THE STRUCTURE OF A WEB PAGE SEMANTICALLY AND ORIGINALLY INCLUDED CUES FOR THE APPEARANCE OF THE DOCUMENT.
